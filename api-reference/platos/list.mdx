## `GET /api/platos`

Returns all dishes that have not been soft-deleted (i.e. where `borrado_en` is `null`).

**Authentication:** Bearer JWT required  
**Required role:** any authenticated user

## Request

No parameters or body required.

## Response

### Success (200)

An array of `platos` objects.

<ResponseField name="[].id" type="string">
  Unique dish ID (nanoid, 10 characters)
</ResponseField>

<ResponseField name="[].nombre" type="string">
  Dish name
</ResponseField>

<ResponseField name="[].precio" type="string">
  Dish price as a decimal string (e.g. `"25.00"`)
</ResponseField>

<ResponseField name="[].creado_en" type="string">
  Creation timestamp (ISO 8601)
</ResponseField>

<ResponseField name="[].actualizado_en" type="string">
  Last update timestamp (ISO 8601)
</ResponseField>

<ResponseField name="[].borrado_en" type="string | null">
  Always `null` in this response — soft-deleted dishes are excluded
</ResponseField>

### Error responses

| Status | Description |
|---|---|
| 401 | Unauthorized — missing or invalid JWT |

## Example

```bash
curl https://api.example.com/api/platos \
  -H "Authorization: Bearer <token>"
```
